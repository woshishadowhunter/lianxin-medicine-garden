## Positioning

The project is not a generic todo app or a simple demo. It is a complete community activity operations system with:

- family binding,
- photo-backed submissions,
- administrator review,
- offline retry,
- community statistics,
- exportable reports,
- deployment and database documentation.

## Community Value

The strongest community value is reuse by small organizations that cannot afford a custom system but still need auditable records and administrator workflows.
