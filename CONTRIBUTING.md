# Contributing

Thanks for your interest in improving Lianxin Medicine Garden.

This project is maintained as a practical WeChat Mini Program template for community activity records. Contributions should stay focused on deployment reliability, Mini Program compatibility, data safety, administrator workflows, and documentation quality.

## Good First Contributions

- Improve setup and deployment documentation
- Add screenshots or short walkthroughs for operators
- Add smoke tests for cloud functions
- Improve error messages in family binding and submission flows
- Review cloud database permission assumptions
- Add sample data that does not contain real personal information

## Development Notes

1. Import the repository in WeChat Developer Tools.
2. Use your own Mini Program AppID and WeChat Cloud environment ID.
3. Never commit real cloud IDs, secrets, administrator passwords, or private project config.
4. Keep changes small and focused.
5. Update documentation when behavior or deployment steps change.

## Pull Request Checklist

- The change is scoped to one clear problem.
- No secrets, private IDs, or real user data are committed.
- JavaScript files touched by the change pass syntax checks where practical.
- Documentation is updated when setup, permissions, or behavior changes.
